تفسير الآية رقم 65 من سورة المائدة

عَقَّب نهيهم وذمّهم بدعوتهم للخير بطريقة التّعريض إذ جاء بحرف الإمتناع فقال : { وَلَوْ أنّ أهل الكتاب آمنوا واتَّقوا } ، والمراد اليهود . والمراد بقوله : { آمَنوا } الإيمان بمحمّد صلى الله عليه وسلم وفي الحديث : « اثنان يُؤتَوْن أجرهم مرّتين : رجل من أهل الكتاب آمن بنبيّه ثُمّ آمن بي ( أي عندما بلغته الدّعوة المحمّديّة ) فله أجران ، ورجل كانت له جارية فأدّبها فأحْسَن تأديبها وعلَّمها ثمّ أعْتَقَها فتزوّجها فله أجران »
واللام في قوله : { لكفّرنا عنهم } وقوله { ولأدخلناها } لام تأكيد يكثر وقوعها في جواب ( لَو ) إذا كان فعلاً ماضياً مثبتاً لتأكيد تحقيق التلازم بين شرط ( لو ) وجوابها ، ويكثر أن يجرّد جواب لو عن اللام ، كما سيأتي عند قوله تعالى : { لو نشاء جعلناه أجاجاً } في سورة الواقعة ( 70 ) .
