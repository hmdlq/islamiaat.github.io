تفسير الآية رقم 39 من سورة الذاريات

فَتَوَلَّى بِرُكْنِهِ وَقَالَ سَاحِرٌ أَوْ مَجْنُونٌ (39(وقوله : { فتولى بركنه } تمثيل لهيئة رفضه دعوة موسى بهيئة المنصرف عن شخص . وبإيرادِ قوله : { بركنه } تَمَّ التمثيل ولولاه لكان قوله : { تولى } مجرد استعارة .
والباء للملابسة ، أي ملابساً ركنه كما في قوله : { أعرض ونأى بجانبه } [ الإسراء : 83 ] .
