تفسير الآية رقم 44 من سورة الشورى

يخبر تعالى أنه المنفرد بالهداية والإضلال، وأنه { مَنْ يُضْلِلِ اللَّهُ } بسبب ظلمه { فَمَا لَهُ مِنْ وَلِيٍّ مِنْ بَعْدِهِ } يتولى أمره ويهديه.
{ وَتَرَى الظَّالِمِينَ لَمَّا رَأَوُا الْعَذَابَ } مرأى ومنظرا فظيعا، صعبا شنيعا، يظهرون الندم العظيم، والحزن على ما سلف منهم، و { يَقُولُونَ هَلْ إِلَى مَرَدٍّ مِنْ سَبِيلٍ } أي: هل لنا طريق أو حيلة إلى رجوعنا إلى الدنيا، لنعمل غير الذي كنا نعمل، وهذا طلب للأمر المحال الذي لا يمكن.
