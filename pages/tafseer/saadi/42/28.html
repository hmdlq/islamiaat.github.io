تفسير الآية رقم 28 من سورة الشورى

{ وَهُوَ الَّذِي يُنَزِّلُ الْغَيْثَ } أي: المطر الغزير الذي به يغيث البلاد والعباد، { مِنْ بَعْدِ مَا قَنَطُوا } وانقطع عنهم مدة ظنوا أنه لا يأتيهم، وأيسوا وعملوا لذلك الجدب أعمالا، فينزل الله الغيث { وَيَنْشُرُ } به { رَحْمَتَهُ } من إخراج الأقوات للآدميين وبهائمهم، فيقع عندهم موقعا عظيما، ويستبشرون بذلك ويفرحون. { وَهُوَ الْوَلِيُّ } الذي يتولى عباده بأنواع التدبير، ويتولى القيام بمصالح دينهم ودنياهم. { الْحَمِيدُ } في ولايته وتدبيره، الحميد على ما له من الكمال، وما أوصله إلى خلقه من أنواع الإفضال.
