تفسير الآية رقم 45 من سورة الشورى

{ وَتَرَاهُمْ يُعْرَضُونَ عَلَيْهَا } أي: على النار { خَاشِعِينَ مِنَ الذُّلِّ } أي: ترى أجسامهم خاشعة للذل الذي في قلوبهم، { يَنْظُرُونَ مِنْ طَرْفٍ خَفِيٍّ } أي: ينظرون إلى النار مسارقة وشزرا، من هيبتها وخوفها.
{ وَقَالَ الَّذِينَ آمَنُوا } حيث ظهرت عواقب الخلق، وتبين أهل الصدق من غيرهم: { إِنَّ الْخَاسِرِينَ } على الحقيقة { الَّذِينَ خَسِرُوا أَنْفُسَهُمْ وَأَهْلِيهِمْ يَوْمَ الْقِيَامَةِ } حيث فوتوا أنفسهم جزيل الثواب، وحصلوا على أليم العقاب وفرق بينهم وبين أهليهم، فلم يجتمعوا بهم، آخر ما عليهم. { أَلَا إِنَّ الظَّالِمِينَ } أنفسهم بالكفر والمعاصي { فِي عَذَابٍ مُقِيمٍ } أي: في سوائه ووسطه، منغمرين لا يخرجون منه أبدا، ولا يفتر عنهم وهم فيه مبلسون.
