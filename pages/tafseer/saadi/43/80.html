تفسير الآية رقم 80 من سورة الزخرف

{ أَمْ يَحْسَبُونَ } بجهلهم وظلمهم { أَنَّا لَا نَسْمَعُ سِرَّهُمْ } الذي لم يتكلموا به، بل هو سر في قلوبهم { وَنَجْوَاهُمْ } أي: كلامهم الخفي الذي يتناجون به، أي: فلذلك أقدموا على المعاصي، وظنوا أنها لا تبعة لها ولا مجازاة على ما خفي منها.
فرد اللّه عليهم بقوله: { بَلَى } أي: إنا نعلم سرهم ونجواهم، { وَرُسُلُنَا } الملائكة الكرام، { لَدَيْهِمْ يَكْتُبُونَ } كل ما عملوه، وسيحفظ ذلك عليهم، حتى يردوا القيامة، فيجدوا ما عملوا حاضرا، ولا يظلم ربك أحدا.
