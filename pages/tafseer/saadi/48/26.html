تفسير الآية رقم 26 من سورة الفتح

يقول تعالى: { إِذْ جَعَلَ الَّذِينَ كَفَرُوا فِي قُلُوبِهِمُ الْحَمِيَّةَ حَمِيَّةَ الْجَاهِلِيَّةِ } حيث أنفوا من كتابة { بسم الله الرحمن الرحيم } وأنفوا من دخول رسول الله صلى الله عليه وسلم والمؤمنين إليهم في تلك السنة، لئلا يقول الناس: { دخلوا مكة قاهرين لقريش } وهذه الأمور ونحوها من أمور الجاهلية، لم تزل في قلوبهم حتى أوجبت لهم ما أوجبت من كثير من المعاصي، { فَأَنْزَلَ اللَّهُ سَكِينَتَهُ عَلَى رَسُولِهِ وَعَلَى الْمُؤْمِنِينَ } فلم يحملهم الغضب على مقابلة المشركين بما قابلوهم به، بل صبروا لحكم الله، والتزموا الشروط التي فيها تعظيم حرمات الله ولو كانت ما كانت، ولم يبالوا بقول القائلين، ولا لوم اللائمين.
{ وَأَلْزَمَهُمْ كَلِمَةَ التَّقْوَى } وهي { لا إله إلا الله } وحقوقها، ألزمهم القيام بها، فالتزموها وقاموا بها، { وَكَانُوا أَحَقَّ بِهَا } من غيرهم { و } كانوا { أهلها } الذين استأهلوها لما يعلم الله عندهم وفي قلوبهم من الخير، ولهذا قال: { وَكَانَ اللَّهُ بِكُلِّ شَيْءٍ عَلِيمًا }
