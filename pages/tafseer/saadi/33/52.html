تفسير الآية رقم 52 من سورة الأحزاب

وهذا شكر من اللّه، الذي لم يزل شكورًا، لزوجات رسوله، رضي اللّه عنهن، حيث اخترن اللّه ورسوله، والدار الآخرة، أن رحمهن، وقصر رسوله عليهن فقال: { لَا يَحِلُّ لَكَ النِّسَاءُ مِنْ بَعْدُ } زوجاتك الموجودات { وَلَا أَنْ تَبَدَّلَ بِهِنَّ مِنْ أَزْوَاجٍ } أي: ولا تطلق بعضهن، فتأخذ بدلها.
فحصل بهذا، أمنهن من الضرائر، ومن الطلاق، لأن اللّه قضى أنهن زوجاته في الدنيا والآخرة، لا يكون بينه وبينهن فرقة.
{ وَلَوْ أَعْجَبَكَ حُسْنُهُنَّ } أي: حسن غيرهن، فلا يحللن لك { إِلَّا مَا مَلَكَتْ يَمِينُكَ } أي: السراري، فذلك جائز لك، لأن المملوكات، في كراهة الزوجات، لسن بمنزلة الزوجات، في الإضرار للزوجات. { وَكَانَ اللَّهُ عَلَى كُلِّ شَيْءٍ رَقِيبًا } أي: مراقبًا للأمور، وعالمًا بما إليه تؤول، وقائمًا بتدبيرها على أكمل نظام، وأحسن إحكام.
