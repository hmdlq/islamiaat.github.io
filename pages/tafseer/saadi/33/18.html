تفسير الآية رقم 18 من سورة الأحزاب

ثم توَّعد تعالى المخذلين المعوقين، وتهددهم فقال: { قَدْ يَعْلَمُ اللَّهُ الْمُعَوِّقِينَ مِنْكُمْ } عن الخروج، لمن [لم] يخرجوا { وَالْقَائِلِينَ لِإِخْوَانِهِمْ } الذين خرجوا: { هَلُمَّ إِلَيْنَا } أي: ارجعوا، كما تقدم من قولهم: { يَا أَهْلَ يَثْرِبَ لَا مُقَامَ لَكُمْ فَارْجِعُوا }
وهم مع تعويقهم وتخذيلهم { وَلَا يَأْتُونَ الْبَأْسَ } أي: القتال والجهاد بأنفسهم { إِلَّا قَلِيلًا } فهم أشد الناس حرصًا على التخلف، لعدم الداعي لذلك، من الإيمان والصبر، ووجود المقتضى للجبن، من النفاق، وعدم الإيمان.
