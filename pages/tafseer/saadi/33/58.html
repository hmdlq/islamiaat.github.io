تفسير الآية رقم 58 من سورة الأحزاب

وإن كانت أذية المؤمنين عظيمة، وإثمها عظيمًا، ولهذا قال فيها: { وَالَّذِينَ يُؤْذُونَ الْمُؤْمِنِينَ وَالْمُؤْمِنَاتِ بِغَيْرِ مَا اكْتَسَبُوا } أي: بغير جناية منهم موجبة للأذى { فَقَدِ احْتَمَلُوا } على ظهورهم { بُهْتَانًا } حيث آذوهم بغير سبب { وَإِثْمًا مُبِينًا } حيث تعدوا عليهم، وانتهكوا حرمة أمر اللّه باحترامها.
ولهذا كان سب آحاد المؤمنين، موجبًا للتعزير، بحسب حالته وعلو مرتبته، فتعزير من سب الصحابة أبلغ، وتعزير من سب العلماء، وأهل الدين، أعظم من غيرهم.
