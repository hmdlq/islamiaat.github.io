تفسير الآية رقم 44 من سورة إبراهيم

يقول تعالى لنبيه محمد صلى الله عليه وسلم: { وَأَنْذِرِ النَّاسَ يَوْمَ يَأْتِيهِمُ الْعَذَابُ } أي: صف لهم صفة تلك الحال وحذرهم من الأعمال الموجبة للعذاب الذي حين يأتي في شدائده وقلاقله، { فَيَقُولُ الَّذِينَ ظَلَمُوا } بالكفر والتكذيب وأنواع المعاصي نادمين على ما فعلوا سائلين للرجعة في غير وقتها، { رَبَّنَا أَخِّرْنَا إِلَى أَجَلٍ قَرِيبٍ } أي: ردَّنا إلى الدنيا فإنا قد أبصرنا، { نُجِبْ دَعْوَتَكَ } والله يدعو إلى دار السلام { وَنَتَّبِعِ الرُّسُلَ } وهذا كله لأجل التخلص من العذاب الأليم وإلا فهم كذبة في هذا الوعد { ولو ردوا لعادوا لما نهوا عنه } ولهذا يوبخون ويقال لهم: { أَوَلَمْ تَكُونُوا أَقْسَمْتُمْ مِنْ قَبْلُ مَا لَكُمْ مِنْ زَوَالٍ } عن الدنيا وانتقال إلى الآخرة، فها قد تبين حنثكم في إقسامكم، وكذبكم فيما تدعون
