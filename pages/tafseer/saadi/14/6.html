تفسير الآية رقم 6 من سورة إبراهيم

فإنه يستدل بأيامه على كمال قدرته وعميم إحسانه، وتمام عدله وحكمته، ولهذا امتثل موسى عليه السلام أمر ربه، فذكرهم نعم الله فقال: { اذْكُرُوا نِعْمَةَ اللَّهِ عَلَيْكُمْ } أي: بقلوبكم وألسنتكم. { إِذْ أَنْجَاكُمْ مِنْ آلِ فِرْعَوْنَ يَسُومُونَكُمْ } أي: يولونكم { سُوءَ الْعَذَابِ } أي: أشده وفسر ذلك بقوله: { وَيُذَبِّحُونَ أَبْنَاءَكُمْ وَيَسْتَحْيُونَ نِسَاءَكُمْ } أي: يبقونهن فلا يقتلونهن، { وَفِي ذَلِكُمْ } الإنجاء { بَلَاءٌ مِنْ رَبِّكُمْ عَظِيمٌ } أي: نعمة عظيمة، أو وفي ذلكم العذاب الذي ابتليتم به من فرعون وملئه ابتلاء من الله عظيم لكم، لينظر هل تصبرون أم لا؟
