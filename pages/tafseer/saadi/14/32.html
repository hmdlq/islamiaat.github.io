تفسير الآية رقم 32 من سورة إبراهيم

يخبر تعالى: أنه وحده { الَّذِي خَلَقَ السَّمَاوَاتِ وَالْأَرْضَ } على اتساعهما وعظمهما، { وَأَنْزَلَ مِنَ السَّمَاءِ مَاءً } وهو: المطر الذي ينزله الله من السحاب، { فَأَخْرَجَ } بذلك الماء { مِنَ الثَّمَرَاتِ } المختلفة الأنواع { رِزْقًا لَكُمْ } ورزقا لأنعامكم { وَسَخَّرَ لَكُمُ الْفُلْكَ } أي: السفن والمراكب.
{ لِتَجْرِيَ فِي الْبَحْرِ بِأَمْرِهِ } فهو الذي يسر لكم صنعتها وأقدركم عليها، وحفظها على تيار الماء لتحملكم، وتحمل تجاراتكم، وأمتعتكم إلى بلد تقصدونه.
{ وَسَخَّرَ لَكُمُ الْأَنْهَارَ } لتسقي حروثكم وأشجاركم وتشربوا منها.
