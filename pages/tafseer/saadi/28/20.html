تفسير الآية رقم 20 من سورة القصص

وقيض اللّه ذلك الرجل الناصح، وبادرهم إلى الإخبار لموسى بما اجتمع عليه رَأْيُ ملئهم. فقال: { وَجَاءَ رَجُلٌ مِنْ أَقْصَى الْمَدِينَةِ يَسْعَى } أي: ركضا على قدميه من نصحه لموسى، وخوفه أن يوقعوا به، قبل أن يشعر، فـ { قَالَ يَا مُوسَى إِنَّ الْمَلَأَ يَأْتَمِرُون } أي: يتشاورون فيك { لِيَقْتُلُوكَ فَاخْرُجْ } عن المدينة { إِنِّي لَكَ مِنَ النَّاصِحِينَ } فامتثل نصحه.
