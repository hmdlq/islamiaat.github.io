تفسير الآية رقم 30 من سورة الأنعام

أي: { وَلَوْ تَرَى } الكافرين { إِذْ وُقِفُوا عَلَى رَبِّهِمْ } لرأيت أمرا عظيما، وهَوْلًا جسيما، { قَالَ } لهم موبخا ومقرعا: { أَلَيْسَ هَذَا } الذي ترون من العذاب { بِالْحَقِّ قَالُوا بَلَى وَرَبِّنَا } فأقروا، واعترفوا حيث لا ينفعهم ذلك، { قَالَ فَذُوقُوا الْعَذَابَ بِمَا كُنْتُمْ تَكْفُرُونَ }
