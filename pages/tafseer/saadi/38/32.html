تفسير الآية رقم 32 من سورة ص

فقال ندما على ما مضى منه، وتقربا إلى اللّه بما ألهاه عن ذكره، وتقديما لحب اللّه على حب غيره: { إِنِّي أَحْبَبْتُ حُبَّ الْخَيْرِ } وضمن { أحببت } معنى { آثرت } أي: آثرت حب الخير، الذي هو المال عموما، وفي هذا الموضع المراد الخيل { عَنْ ذِكْرِ رَبِّي حَتَّى تَوَارَتْ بِالْحِجَابِ }
