تفسير الآية رقم 25 من سورة فصلت

أي: وقضينا لهؤلاء الظالمين الجاحدين للحق { قُرَنَاءَ } من الشياطين، كما قال تعالى: { أَلَمْ تَرَ أَنَّا أَرْسَلْنَا الشَّيَاطِينَ عَلَى الْكَافِرِينَ تَؤُزُّهُمْ أَزًّا } أي تزعجهم إلى المعاصي وتحثهم عليها، بسبب ما زينوا { لَهُمْ مَا بَيْنَ أَيْدِيهِمْ وَمَا خَلْفَهُمْ } فالدنيا زخرفوها بأعينهم، ودعوهم إلى لذاتها وشهواتها المحرمة حتى افتتنوا، فأقدموا على معاصي اللّه، وسلكوا ما شاءوا من محاربة اللّه ورسله والآخرة بَعّدُوها عليهم وأنسوهم ذكرها، وربما أوقعوا عليهم الشُّبه، بعدم وقوعها، فترحَّل خوفها من قلوبهم، فقادوهم إلى الكفر، والبدع، والمعاصي.
وهذا التسليط والتقييض من اللّه للمكذبين الشياطين، بسبب إعراضهم عن ذكر اللّه وآياته، وجحودهم الحق كما قال تعالى: { وَمَنْ يَعْشُ عَنْ ذِكْرِ الرَّحْمَنِ نُقَيِّضْ لَهُ شَيْطَانًا فَهُوَ لَهُ قَرِينٌ وَإِنَّهُمْ لَيَصُدُّونَهُمْ عَنِ السَّبِيلِ وَيَحْسَبُونَ أَنَّهُمْ مُهْتَدُونَ }
{ وَحَقَّ عَلَيْهِمُ الْقَوْلُ } أي: وجب عليهم، ونزل القضاء والقدر بعذابهم { فِي } جملة { أُمَمٍ قَدْ خَلَتْ مِنْ قَبْلِهِمْ مِنَ الْجِنِّ وَالْإِنْسِ إِنَّهُمْ كَانُوا خَاسِرِينَ } لأديانهم وآخرتهم، ومن خسر، فلا بد أن يذل ويشقى ويعذب.
