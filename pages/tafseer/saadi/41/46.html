تفسير الآية رقم 46 من سورة فصلت

{ مَنْ عَمِلَ صَالِحًا } وهو العمل الذي أمر اللّه به، ورسوله { فَلِنَفْسِهِ } نفعه وثوابه في الدنيا والآخرة { وَمَنْ أَسَاءَ فَعَلَيْهَا } ضرره وعقابه، في الدنيا والآخرة، وفي هذا، حثٌّ على فعل الخير، وترك الشر، وانتفاع العاملين، بأعمالهم الحسنة، وضررهم بأعمالهم السيئة، وأنه لا تزر وازرة وزر أخرى. { وَمَا رَبُّكَ بِظَلَّامٍ لِلْعَبِيدِ } فَيُحمِّل أحدًا فوق سيئاتهم.
