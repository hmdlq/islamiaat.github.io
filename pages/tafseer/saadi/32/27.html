تفسير الآية رقم 27 من سورة السجدة

{ أَوَلَمْ يَرَوْا } بأبصارهم نعمتنا، وكمال حكمتنا { أَنَّا نَسُوقُ الْمَاءَ إِلَى الْأَرْضِ الْجُرُزِ } التي لا نبات فيها، فيسوق اللّه المطر، الذي لم يكن قبل موجودًا فيها، فيفرغه فيها، من السحاب، أو من الأنهار. { فَنُخْرِجُ بِهِ زَرْعًا } أي: نباتًا، مختلف الأنواع { تَأْكُلُ مِنْهُ أَنْعَامُهُمْ } وهو نبات البهائم { وَأَنْفُسهمْ } وهو طعام الآدميين.
{ أَفَلَا يُبْصِرُونَ } تلك المنة، التي أحيا اللّه بها البلاد والعباد، فيستبصرون فيهتدون بذلك البصر، وتلك البصيرة، إلى الصراط المستقيم، ولكن غلب عليهم العمى، واستولت عليهم الغفلة، فلم يبصروا في ذلك، بصر الرجال، وإنما نظروا إلى ذلك، نظر الغفلة، ومجرد العادة، فلم يوفقوا للخير.
