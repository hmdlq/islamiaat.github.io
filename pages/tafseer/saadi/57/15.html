تفسير الآية رقم 15 من سورة الحديد

{ فَالْيَوْمَ لَا يُؤْخَذُ مِنْكُمْ فِدْيَةٌ وَلَا مِنَ الَّذِينَ كَفَرُوا } فلو افتديتم بمثل الأرض ذهبا ومثله معه، لما تقبل منكم، { مَأْوَاكُمُ النَّارُ } أي: مستقركم، { هِيَ مَوْلَاكُمْ } التي تتولاكم وتضمكم إليها، { وَبِئْسَ الْمَصِيرُ } النار.
[قال تعالى:] { وَأَمَّا مَنْ خَفَّتْ مَوَازِينُهُ فَأُمُّهُ هَاوِيَةٌ وَمَا أَدْرَاكَ مَا هِيَهْ نَارٌ حَامِيَةٌ }
