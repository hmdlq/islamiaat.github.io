تفسير الآية رقم 37 من سورة هود

{ وَاصْنَعِ الْفُلْكَ بِأَعْيُنِنَا وَوَحْيِنَا } أي: بحفظنا، ومرأى منا, وعلى مرضاتنا، { وَلَا تُخَاطِبْنِي فِي الَّذِينَ ظَلَمُوا } أي: لا تراجعني في إهلاكهم، { إِنَّهُمْ مُغْرَقُونَ } أي: قد حق عليهم القول، ونفذ فيهم القدر.
