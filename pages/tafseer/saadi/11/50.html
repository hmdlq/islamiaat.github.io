تفسير الآية رقم 50 من سورة هود

إلى آخر القصة أي: { و } أرسلنا { إِلَى عَادٍ } وهم القبيلة المعروفة في الأحقاف, من أرض اليمن، { أَخَاهُمْ } في النسب { هُودًا } ليتمكنوا من الأخذ عنه والعلم بصدقه.
فـ { قَالَ } لهم { يَا قَوْمِ اعْبُدُوا اللَّهَ مَا لَكُمْ مِنْ إِلَهٍ غَيْرُهُ إِنْ أَنْتُمْ إِلَّا مُفْتَرُونَ } أي: أمرهم بعبادة الله وحده، ونهاهم عما هم عليه، من عبادة غير الله، وأخبرهم أنهم قد افتروا على الله الكذب في عبادتهم لغيره, وتجويزهم لذلك، ووضح لهم وجوب عبادة الله، وفساد عبادة ما سواه.
