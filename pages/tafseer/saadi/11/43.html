تفسير الآية رقم 43 من سورة هود

فـ { قَالَ } ابنه، مكذبا لأبيه أنه لا ينجو إلا من ركب معه السفينة.
{ سَآوِي إِلَى جَبَلٍ يَعْصِمُنِي مِنَ الْمَاءِ } أي: سأرتقي جبلا، أمتنع به من الماء، فـ { قَالَ } نوح: { لَا عَاصِمَ الْيَوْمَ مِنْ أَمْرِ اللَّهِ إِلَّا مَنْ رَحِمَ } فلا يعصم أحدا، جبل ولا غيره، ولو تسبب بغاية ما يمكنه من الأسباب، لما نجا إن لم ينجه الله. { وَحَالَ بَيْنَهُمَا الْمَوْجُ فَكَانَ } الابن { مِنَ الْمُغْرَقِينَ }
