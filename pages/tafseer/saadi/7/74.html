تفسير الآية رقم 74 من سورة الأعراف

{ وَاذْكُرُوا إِذْ جَعَلَكُمْ خُلَفَاءَ } في الأرض تتمتعون بها وتدركون مطالبكم { مِنْ بَعْدِ عَادٍ } الذين أهلكهم اللّه، وجعلكم خلفاء من بعدهم، { وَبَوَّأَكُمْ فِي الْأَرْضِ } أي: مكن لكم فيها، وسهل لكم الأسباب الموصلة إلى ما تريدون وتبتغون { تَتَّخِذُونَ مِنْ سُهُولِهَا قُصُورًا } أي: من الأراضي السهلة التي ليست بجبال، تتخذون فيها القصور العالية والأبنية الحصينة، { وَتَنْحِتُونَ الْجِبَالَ بُيُوتًا } كما هو مشاهد إلى الآن من أعمالهم التي في الجبال، من المساكن والحجر ونحوها، وهي باقية ما بقيت الجبال، { فَاذْكُرُوا آلَاءَ اللَّهِ } أي: نعمه، وما خولكم من الفضل والرزق والقوة، { وَلَا تَعْثَوْا فِي الْأَرْضِ مُفْسِدِينَ } أي: لا تخربوا الأرض بالفساد والمعاصي، فإن المعاصي تدع الديار العامرة بلاقع، وقد أخلت ديارهم منهم، وأبقت مساكنهم موحشة بعدهم.
