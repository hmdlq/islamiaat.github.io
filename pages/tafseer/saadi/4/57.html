تفسير الآية رقم 57 من سورة النساء

{ وَالَّذِينَ آمَنُوا } أي: بالله وما أوجب الإيمانَ به { وَعَمِلُوا الصَّالِحَاتِ } من الواجبات والمستحبات { سَنُدْخِلُهُمْ جَنَّاتٍ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ خَالِدِينَ فِيهَا أَبَدًا لَهُمْ فِيهَا أَزْوَاجٌ مُطَهَّرَةٌ } أي: من الأخلاق الرذيلة، والخلْق الذميم، ومما يكون من نساء الدنيا من كل دنس وعيب { وَنُدْخِلُهُمْ ظِلًّا ظَلِيلًا }
