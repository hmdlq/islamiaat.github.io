تفسير الآية رقم 47 من سورة العنكبوت

أي: { وَكَذَلِكَ أَنْزَلْنَا إِلَيْكَ } يا محمد، هذا { الْكِتَاب } الكريم، المبين كل نبأ عظيم، الداعي إلى كل خلق فاضل، وأمر كامل، المصدق للكتب السابقة، المخبر به الأنبياء الأقدمون.
{ فَالَّذِينَ آتَيْنَاهُمُ الْكِتَابَ } فعرفوه حق معرفته، ولم يداخلهم حسد وهوى.
{ يُؤْمِنُونَ بِهِ } لأنهم تيقنوا صدقه، بما لديهم من الموافقات، وبما عندهم من البشارات، وبما تميزوا به من معرفة الحسن والقبيح، والصدق والكذب.
{ وَمِنْ هَؤُلَاءِ } الموجودين { مَنْ يُؤْمِنُ بِهِ } إيمانا عن بصيرة، لا عن رغبته ولا رهبته. { وَمَا يَجْحَدُ بِآيَاتِنَا إِلَّا الْكَافِرُونَ } الذين دأبهم الجحود للحق والعناد له. وهذا حصر لمن كفر به، أنه لا يكون من أحد قصده متابعة الحق، وإلا، فكل من له قصد صحيح، فإنه لا بد أن يؤمن به، لما اشتمل عليه من البينات، لكل من له عقل، أو ألقى السمع وهو شهيد.
