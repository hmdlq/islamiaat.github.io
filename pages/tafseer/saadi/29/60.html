تفسير الآية رقم 60 من سورة العنكبوت

أي: الباري تبارك وتعالى، قد تكفل بأرزاق الخلائق كلهم، قويهم وعاجزهم، فكم { مِنْ دَابَّةٍ } في الأرض، ضعيفة القوى، ضعيفة العقل. { لَا تَحْمِلُ رِزْقَهَا } ولا تدخره، بل لم تزل، لا شيء معها من الرزق، ولا يزال اللّه يسخر لها الرزق، في كل وقت بوقته.
{ اللَّهُ يَرْزُقُهَا وَإِيَّاكُمْ } فكلكم عيال اللّه، القائم برزقكم، كما قام بخلقكم وتدبيركم، { وَهُوَ السَّمِيعُ الْعَلِيمُ } فلا يخفى عليه خافية، ولا تهلك دابة من عدم الرزق بسبب أنها خافية عليه.
كما قال تعالى: { وَمَا مِنْ دَابَّةٍ فِي الْأَرْضِ إِلَّا عَلَى اللَّهِ رِزْقُهَا وَيَعْلَمُ مُسْتَقَرَّهَا وَمُسْتَوْدَعَهَا كُلٌّ فِي كِتَابٍ مُبِينٍ }
